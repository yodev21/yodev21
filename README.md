## 次のキャリアで目指すこと

クラウドリードとして、プロダクト開発にクラウドの観点から意見・提案ができる存在を目指しています。

開発チームを支える基盤づくりに関心があり、複数のプロダクトを横断しながら、クラウド・インフラ領域で頼られる存在を目指します。

軸となる技術領域は Terraform / IaC と Observability（監視・ログ・トレーシング）。これまでの経験で培った「運用を見据えた設計」「運用データに基づく改善」を武器に、発信力と技術の深さを両輪で磨いていきます。
