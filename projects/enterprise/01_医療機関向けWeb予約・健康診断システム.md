## 案件名
医療機関向けWeb予約・健康診断システム 導入・運用支援（オンプレミス）

## 期間
2016年 ～ 2020年

## 体制
1～5名（顧客先常駐／案件規模により変動）

## 役割
導入SE / カスタマーサポート（メイン） / インフラ構築・アプリ改修（一部）

## 技術スタック
- Windows Server / CentOS
- Java / VB.Net / VBA / SQL Server

## プロジェクト概要
医療機関向けストレスチェック・健康診断Web予約システムの導入および運用支援を担当。主に導入作業やカスタマーサポートを担当し、一部の期間でサーバー構築やアプリケーション改修にも携わった。

## 担当業務

### 導入・サポート
- 顧客先でのシステム導入・初期設定
- 操作説明・トレーニング
- 問い合わせ対応、障害調査、トラブルシュート
- マスタ整備、パッチ適用、リリース作業

### 開発・インフラ構築
- オンプレミスサーバー環境構築（Linux/Windows）
- 軽微な機能改修およびデータ連携対応

## 主な実績
- CentOS上でのWeb予約システムを主担当として構築から運用保守・追加改修まで推進
- 健康診断システムとのデータ連携機能を実装
- 複数の医療機関への導入を経験
