## フェーズ

## 期間
2025年10月 ～ 現在

## 体制

クラウド基盤構築・運用保守 / バックエンド・アプリ機能開発

## 稼働率
50%

## 技術スタック
- Google Cloud（Cloud Run / Cloud Run Jobs / Pub/Sub / Cloud Tasks / Cloud SQL / Memorystore / Cloud Storage / Cloud Armor / Load Balancer / DNS / Cloud Build / Secret Manager / IAM / VPC / NAT / Compute Engine / Vertex AI）

## プロジェクト概要
既に本番稼働している海外送金サービスに対する継続的な追加機能開発および運用保守。クラウド基盤整備からアプリケーション開発まで横断的に担当し、サービスの信頼性・拡張性向上を推進。

## 担当業務
- Google Cloud におけるクラウド基盤構築および運用保守
- Cloud Run / Pub/Sub / Cloud Tasks を用いた非同期処理アーキテクチャ設計
- バックエンドAPIおよびアプリケーション機能開発（Go）
- フロントエンド開発支援（Next.js/React）
- CI/CD整備および運用改善

## 成果
- Cloud Run + Pub/Sub を用いたキューイング基盤を設計・構築し、非同期処理によるスケーラブルなメッセージ処理を実現
- インフラ構築からアプリケーション実装まで一貫して担当
