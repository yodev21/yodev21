## 案件名
顧客管理システム開発

## フェーズ
- 設計 / 構築：2023年2月 ～ 2023年12月（稼働率20～50%）
- 運用保守 / 技術支援：2024年2月 ～ 2024年5月（稼働率20%）

## 体制
複数名体制

## 役割
クラウド基盤構築 / CI/CD整備 / 運用技術支援

## 技術スタック
- AWS（Amazon ECS / AWS Lambda / Amazon Aurora / Amazon S3 / Application Load Balancer / AWS CodeBuild / Amazon OpenSearch Service）
- Python
- Vue.js

## プロジェクト概要
AWS上に新規構築された顧客管理システムのクラウド基盤設計・構築および運用フェーズでの技術支援を担当。インフラ標準化、CI/CD整備、検索基盤の実装、安定運用体制の確立を推進。

## 担当業務（設計 / 構築フェーズ）
- Amazon ECS / AWS Lambda / Amazon Aurora / Amazon S3 を用いたクラウド基盤の設計・構築
- AWS CodeBuild によるCI/CDパイプライン設計・構築
- Amazon OpenSearch Service を用いた検索基盤の設計・実装支援
- アプリケーション開発支援および環境整備

## 担当業務（運用保守 / 技術支援フェーズ）
- クラウド環境の運用保守および改善対応
- メンバーへの技術支援、設計レビュー、トラブルシュート対応
- 安定運用に向けた運用フロー整備

## 成果
- インフラおよびCI/CDの標準化により開発効率を向上
- 検索基盤の整備により高速なデータ検索機能を実現
- 運用フェーズにおいて安定稼働体制を確立
